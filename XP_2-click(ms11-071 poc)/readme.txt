Instructions for Local Network test

1. Two Windows computers are needed for this PoC:
		ComputerA (any Windows version) and ComputerB (Windows XP)
2. On ComputerA unzip the XP_2-click.zip to C:\XP_2-click
3. On ComputerA make C:\XP_2-click a shared folder named XP_2-click accessible to everyone
4. On ComputerB start Internet Explorer 8 and open \\ComputerA\XP_2-click\test.html then follow the onscreen instructions

Note:
- This PoC can also be executed locally by opening C:\XP_2-click\test.html in IE (when asked allow scripting)
